# Package Info

Asset name: `Borzblade PSX/PS2 Retro Render Toolkit for URP`

Version: `1.0.0`

Unity version tested: `6000.4.1f1`

URP version tested: `17.4.0`

Supported render pipeline: URP

Unsupported render pipelines: Built-in Render Pipeline, HDRP

## Folder Structure

- `Art`: placeholder textures, icons, and previews.
- `Demo`: optional demo content, if supplied.
- `Documentation`: README, quick start, troubleshooting, and package info.
- `Editor`: shader inspectors, windows, tools, and editor-only helpers.
- `Materials`: packaged sample materials and presets, including sprite, water, and outline materials.
- `Runtime`: renderer features and runtime utilities.
- `Samples`: optional material, terrain, foliage, and renderer comparison samples.
- `Settings`: renderer feature materials, presets, and global profiles.
- `Shaders`: core includes, hybrid, terrain, cutout, foliage, unlit, sprite, water, outline, and fullscreen shaders.

## Quick Setup

Open `Tools > Borzblade > Retro Render Toolkit > Open Toolkit`, install the renderer feature, create default materials, optionally install depth fog or screen outline features, then assign the materials in your scene.

For support, use the publisher contact listed on the Asset Store page or package metadata.
